# JobScout AI – Autonomous Career Agent

JobScout AI is an AI-powered autonomous job hunting assistant built for the SMIT Agentic AI Hackathon.
## Video link

https://docs.google.com/videos/d/1ISpYmvoZdGt13Xuv0iDNnS2Qsxj5ov1cF2_OnklDqgQ/edit?usp=sharing

## Features

- Upload CV in PDF format
- AI extracts skills and experience
- Autonomous job search using AI-generated queries
- Real-time market evaluation
- Match scoring and reasoning
- Interactive dashboard interface

## Agentic Workflow

1. User uploads CV
2. AI extracts profile data
3. Agent plans search queries
4. Real-time job search performed
5. Jobs evaluated against profile
6. Ranked matches displayed with reasoning
